
**Option A: Source Code Injection**

1.  **Intercept the Signal**:
    ```bash
    git clone https://github.com/yourusername/AiWhisperer.git
    cd AiWhisperer
    ```

2.  **Establish Environment**:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3.  **Install Neural Maps**:
    ```bash
    pip install -r requirements.txt
    ```

4.  **Engage**:
    ```bash
    python ai_whisperer.py
    ```

**Option B: Compile Binary Artifact (.app)**

1.  **Purge Old Matter**:
    ```bash
    rm -rf dist build
    ```

2.  **Construct Artifact**:
    ```bash
    python setup.py py2app
    ```

3.  **Finalize & Launch**:
    Locate `dist/AiWhisperer.app` and drag it to your Applications sector.
